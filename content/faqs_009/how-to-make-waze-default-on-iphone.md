<h2>Step 2: Select Waze as the Default Navigation App</h2>

<p>The next step is to select Waze as your default navigation app. To do this, open the Settings app on your iPhone and select “General.” Then select “Default Navigation App.” You will then see a list of available navigation apps. Select Waze from the list and confirm your selection.</p>

<h2>Step 3: Set Waze as the Default Maps App</h2>

<p>The final step is to set Waze as the default maps app. To do this, open the Settings app and select “Maps.” Then select “Default Maps App.” You will then see a list of available maps apps. Select Waze from the list and confirm your selection.</p>
