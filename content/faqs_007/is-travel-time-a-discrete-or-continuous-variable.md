<h2>What is a Discrete Variable?</h2>

A discrete variable is one that can only take on a finite number of values. For example, a person's age can only take on a certain number of values, such as 18, 19, 20, and so on. A discrete variable is usually expressed as a whole number or an integer.

<h2>What is a Continuous Variable?</h2>

A continuous variable is one that can take on any value within a given range. For example, a person's height can take on any value between 5 feet and 6 feet, such as 5.5 feet, 5.75 feet, 6.25 feet, and so on. A continuous variable is usually expressed as a decimal value or a fraction.
