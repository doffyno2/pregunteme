<h2>How to Change a Repository Name on GitHub</h2>

<p>The first step in changing a repository name is to log in to your GitHub account. Once you have done so, you will need to navigate to the repository that you wish to rename. On the repository page, you will see a “Settings” button. Click on it.</p>

<p>The next step is to click on the “Rename” button. You will be prompted to enter a new name for the repository. Once you have done so, click the “Rename Repository” button at the bottom of the page.</p>

<p>GitHub will now prompt you to confirm the new repository name. Once you have done so, click the “Rename Repository” button at the bottom of the page again. This will complete the process and the repository will be renamed.</p>
